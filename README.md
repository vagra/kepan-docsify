#1 Kepan Docsify

用支持一百零八级标题的 Markdown 语法来编写佛经科判。

修改现有的 Markdown 语法，不再使用 `#、##、###、####、#####、######` ，改以 `#1、#2、#3、#4、#5、#6……#107、#108` 来代表各级标题，并且从原来的最大六级标题扩展到最大一百零八级标题。

一百零八，是佛教的吉祥数。
